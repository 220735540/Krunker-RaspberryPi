# How to install Krunker
Copy the below and it will do it for you
## Install
```sh
git clone https://github.com/220735540/Krunker-RaspberryPi/
cd "Krunker-Raspberry-Pi/Pi-Ware Install Data"
bash install
```
Enjoy!

## Uninstall Pi-Ware
```sh
cd "Krunker-Raspberry-Pi/Pi-Ware Install Data"
bash uninstall
```
Enjoy Krunker For Raspberry Pi
